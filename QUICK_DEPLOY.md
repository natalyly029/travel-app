# 🚀 ワンクリックデプロイ

## 1️⃣ 下記ボタンをクリック

```
https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2Fnatalyly029%2Ftravel-app&env=NEXT_PUBLIC_SUPABASE_URL,NEXT_PUBLIC_SUPABASE_ANON_KEY,SUPABASE_SERVICE_ROLE_KEY,NEXT_PUBLIC_APP_NAME&envDescription=Supabase+API+credentials&envLink=https%3A%2F%2Fsupabase.com%2Fdashboard%2Fproject%2Fssqifnxhbywabvglnppj%2Fsettings%2Fapi&project-name=travel-app&repository-name=travel-app
```

または、以下を実行（簡単）:

```bash
open "https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2Fnatalyly029%2Ftravel-app"
```

---

## 2️⃣ Vercel が開く

1. **GitHub でログイン** が必要な場合はログイン
2. **Create** をクリック

---

## 3️⃣ 環境変数を入力

下記 4つを入力:

```
NEXT_PUBLIC_SUPABASE_URL
= https://ssqifnxhbywabvglnppj.supabase.co

NEXT_PUBLIC_SUPABASE_ANON_KEY
= [Supabase Dashboard から取得]

SUPABASE_SERVICE_ROLE_KEY
= [Supabase Dashboard から取得]

NEXT_PUBLIC_APP_NAME
= Travel App
```

**Supabase API キー取得先**:
https://supabase.com/dashboard/project/ssqifnxhbywabvglnppj/settings/api

---

## 4️⃣ Deploy をクリック

3-5分待機

---

## 5️⃣ 本番 URL が表示される

```
✅ Production: https://travel-app-xxxx.vercel.app
```

このリンクをブックマーク！
